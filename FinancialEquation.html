Financial Equation

`{((1 + i)^n * PV + (1 + i * S) * ((1 + i)^n - 1)/i * Pmt + FV = 0, if i ne 0), (PV + n * Pmt + FV = 0, if i = 0):}`

`S lArr {(S = 0 larr \mathbf(text(Delayed Payment))), (S = 1 larr \mathbf(text(Prepayment))) :}`

`PV` `lArr` Present Value
`FV` `lArr` Future Value
`Pmt` `lArr` Payment
`n` `lArr` Periods Number
`i` `lArr` Interest Rate

`PV = PV(FV, Pmt) rArr PV = - FV * (1 + i)^-n - (1 + i * S) * (1 - (1 + i)^-n) / i * Pmt`
`FV = FV(PV, Pmt) rArr FV = (- PV - (1 + i * S) * (1 - (1 + i)^-n) / i * Pmt) / (1 + i)^-n`
`Pmt = Pmt(PV, FV) rArr Pmt = (- PV - FV * (1 + i)^-n) * i / ((1 + i * S) * (1 - (1 + i)^-n))`
